# UI/UX 设计规范

## 秘境探索 (HiddenGems) - 视觉与交互设计系统

---

## 1. 设计原则

### 1.1 核心设计理念

| 原则 | 说明 |
|------|------|
| **简洁** | 减少视觉噪音，突出核心内容 |
| **自然** | 贴近旅行主题，营造探索氛围 |
| **智能** | AI推荐突出，但不过度打扰 |
| **信任** | 清晰的信息层级，建立用户信任 |

### 1.2 设计目标

```
用户旅程:
发现 → 探索 → 规划 → 出行 → 分享
 ↓       ↓       ↓       ↓       ↓
惊喜    沉浸    高效    便捷    成就
```

## 5. 动效规范

### 5.1 过渡动画

| 场景 | 动画类型 | 时长 | 缓动曲线 |
|------|----------|------|----------|
| 页面切换 | 共享元素过渡 | 300ms | FastOutSlowIn |
| 卡片点击 | 缩放 | 100ms | Linear |
| 底部抽屉 | 滑入/滑出 | 250ms | Decelerate |
| Toast | 淡入淡出 | 200ms | Linear |
| 加载 | 渐变旋转 | 1000ms | Linear (循环) |
| 点赞 | 缩放+弹跳 | 300ms | Bounce |

### 5.2 手势交互

| 手势 | 场景 | 行为 |
|------|------|------|
| 下拉 | 列表页 | 刷新 |
| 上拉 | 列表页 | 加载更多 |
| 左滑 | 卡片 | 显示操作按钮 |
| 右滑 | 详情页 | 返回 |
| 长按 | 图片 | 保存/分享 |
| 双击 | 图片 | 放大 |

### 5.3 微交互

```
点赞动画:
  1. 心形从空心变为实心
  2. 心形放大至1.3倍
  3. 弹回至1.0倍
  4. 小粒子飘散效果

收藏动画:
  1. 书签图标变实心
  2. 星星粒子环绕
  3. Toast提示"已收藏"

加载动画:
  1. 显示骨架屏
  2. 内容渐入
  3. 骨架屏淡出
```

## 8. 无障碍设计

### 8.1 对比度

- 正常文字: 至少 4.5:1
- 大文字 (18sp+): 至少 3:1
- 图标/图形: 至少 3:1

### 8.2 触控目标

- 最小尺寸: 48dp × 48dp
- 间距: 至少 8dp

### 8.3 语义化

- 为所有图片提供 contentDescription
- 为按钮提供明确的文字标签
- 支持屏幕阅读器
